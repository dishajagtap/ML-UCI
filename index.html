Who We Are

We are the first Artifical Intelligence club at the University of California, Irvine that specializes in the branch of Machine Learning. Artificial Intelligence (AI) is a branch of computer science that deals with building systems that show 'intelligence.' So far, Machine Learning has shown the most promise and is the most common way of developing AI. It is a field that aims to teach computers to learn from examples and perform a task without being explicitly programmed to do it.

What We Do

  • Supervised Learning

    The machine learning task of inferring a function from labeled training data, so that when you have new inputs you can predict the output.

  • Reinforcement Learning

    A subfield of machine learning inspired by behavorial psychology, concerned with how software agents ought to take action in an environment to maximize a reward.

  • Transfer Learning

    A subfield of machine learning where data/knowledge used to solve one problem is then applied to a different but related problem.
